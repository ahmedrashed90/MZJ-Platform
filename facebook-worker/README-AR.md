# MZJ Facebook Transport Worker v1.18.1

هذا الـWorker مسؤول تقنيًا فقط عن:

- التحقق من Meta Webhook واستقبال كل رسائل Messenger والـQuick Replies والـPostbacks والوسائط.
- تمرير الحدث الموحد إلى منصة MZJ.
- إرسال النصوص والأزرار والوسائط بطلب من المنصة.
- منع تكرار الإرسال باستخدام معرف الإرسال الداخلي.

لا يحتوي على أي منطق خاص بالكاش أو التمويل أو خدمة العملاء، ولا ينشئ العميل ولا يوزعه. مسار `/automation` محفوظ للتوافق فقط ويؤكد أن Meta Webhook هو مسار الاستقبال المعتمد، حتى لا تُحسب رسالة العميل مرتين.
