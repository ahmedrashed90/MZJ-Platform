# تقرير الصلاحيات النهائية لكل مستخدم

لا يمكن إصدار جدول نهائي بأسماء المستخدمين من المرفقات الحالية لأن جدول Excel المشار إليه في المتطلبات لم يُرفق.

بعد توفير الجدول، يجب أن تتم المطابقة بالبريد الإلكتروني، ثم تحديث:

- الأدوار العامة.
- الأنظمة المسموحة.
- الدور الخاص بكل نظام.
- الفروع والأقسام.
- نطاق البيانات لكل نظام.
- السماح والمنع الفردي.

تمت إضافة قالب استيراد في `database/user-access-import-template.csv`.
