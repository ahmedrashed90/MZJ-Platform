# تعديل إدخال بيانات مبيعات التمويل — v1.18.0

تم تحويل إدخال بيانات مبيعات التمويل من ثلاث رسائل متتابعة إلى طلب واحد يحتوي على:

- الاسم
- السيارة
- رقم الجوال

يدعم المحرك إدخال البيانات في رسالة واحدة مفصولة بأسطر أو فواصل أو شرطات أو باستخدام أسماء الحقول، كما يدعم استكمال الحقول الناقصة في رسائل لاحقة. عند اكتمال الحقول الثلاثة ينفذ الإجراء النهائي ويرسل رسالة النهاية مباشرة.

التعديل Runtime فقط ولا يتطلب SQL أو Migration ولا يغير فلو مبيعات الكاش أو خدمة العملاء.
