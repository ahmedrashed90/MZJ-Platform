# تقرير التعارضات

## تعارضات المصدر

1. الوصف يطلب جدول مستخدمين Excel، لكن الملف غير موجود ضمن المرفقات؛ لم يتم افتراض المستخدمين أو الأنظمة أو النطاقات الخاصة بهم.
2. الوصف يتعامل مع الأنظمة الأربعة كأنها مدمجة بالكامل، بينما السورس المرفوع يحتوي CRM كاملًا وثلاث صفحات Placeholder للتسويق والعمليات والتراكينج.
3. تمت معالجة التعارض بتطبيق النظام المركزي وحماية الدخول للأنظمة الأربعة، وتطبيق حماية الصفحة والإجراء والـAPI تفصيليًا على الموديول الموجود فعليًا فقط.
4. تم الحفاظ على الأدوار القديمة (`marketing_user`, `operations_user`) مع إضافة الأدوار المسماة في المتطلبات، حتى لا تنكسر العلاقات الحالية.

## قاعدة الحسم

عند التعارض يكون المنع هو الافتراضي، والمنع الفردي يتغلب على السماح والدور. تعطيل النظام يمنع الصلاحيات الفعالة دون حذف الإعدادات المخزنة.
