# CRM v1.15.5 — سير العمل المكتمل وجهات الاتصال

هذه النسخة مبنية مباشرة على النسخة الكاملة v1.15.4، ولا تعتمد على باتش خارجي.

## الحالة النهائية

- الكاش والتمويل: `تم البيع`.
- خدمة العملاء: `تم الانتهاء`.
- عند الوصول للحالة النهائية يُغلق طلب الخدمة، ويختفي العميل من داش بورد المتابعة فقط.
- يظل العميل محفوظًا في قاعدة البيانات والتقارير وسجل جهة الاتصال.

## رسائل غير مصنفة

تعرض فقط المحادثات التي لا يوجد لها طلب خدمة مصنف. بعد اختيار كاش أو تمويل أو خدمة العملاء، يتم إنشاء الطلب والتوزيع وتختفي المحادثة من الصفحة.

## جهات الاتصال

لكل رقم ملف دائم يحتوي على:

- كل سجلات العملاء المرتبطة بالرقم.
- بيانات العميل والحقول الإضافية والملاحظات.
- طلبات الخدمة المفتوحة والمنتهية.
- المحادثات والرسائل الأخيرة.
- سجل الحالات والتحويلات والتوزيع.

الحذف الكامل متاح للإدارة فقط، ويتطلب كتابة رقم الجوال المطابق، ويُستخدم لمسح بيانات الاختبار نهائيًا.

## قاعدة البيانات

زر التعديل يفتح نموذج بيانات العميل داخل صفحة قاعدة البيانات بدون تحميل أو فتح المحادثة.
