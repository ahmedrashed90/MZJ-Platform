# CRM v1.15.8 - تنسيق الواجهات وتسجيل الفويس

تم تنفيذ هذه النسخة مباشرة انطلاقًا من سورس v1.15.7 الكامل، بدون إنشاء طبقة Patch إضافية.

## التعديلات

- إعادة تنسيق صفحة تقييم المناديب KPI مع الحفاظ على نفس المعادلات وقاعدة البيانات.
- فتح نموذج التقييم في مساحة شاشة كاملة.
- استخدام نفس تقسيم ألوان النظام القديم للنسب: أخضر من 80%، أصفر من 50% إلى أقل من 80%، وأحمر لأقل من 50%.
- إبراز أسماء المناديب في جداول الإضافة والتقارير.
- إعادة تنظيم فلاتر صفحة التقارير في صفوف ومجموعات واضحة، وإبراز أسماء العملاء والمناديب.
- تبسيط رأس صفحة سجل عملاء التمويل وإزالة البلوك الملون، وفصل حقول التاريخ والحالة عن البحث.
- حذف صفحة سجل ملكية العملاء ومسارها وواجهة API المخصصة لها، مع استمرار حفظ وعرض تغييرات المسؤول داخل ملف جهة الاتصال.
- إضافة تسجيل صوت من داخل محادثة العميل باستخدام MediaRecorder، ثم رفعه إلى R2 وإرساله كمرفق صوتي عبر نفس مسار المرفقات الحالي.

## ملاحظة الفويس

نجاح التسليم النهائي يعتمد على دعم مرسال وواتساب لنوع الملف الصوتي الذي ينتجه المتصفح. المنصة تحفظ التسجيل وترفعه وترسله عبر نفس Endpoint الحالي بدون تغيير منطق الوركر.
