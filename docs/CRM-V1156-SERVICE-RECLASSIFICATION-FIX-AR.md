# CRM v1.15.6 — إصلاح تغيير القسم وإعادة التوزيع

- كل اختيار صريح من أزرار مرسال: مبيعات الكاش، مبيعات التمويل، خدمة العملاء، يصل للمنصة كتصنيف موثوق.
- إذا اختار العميل نفس القسم، لا يتم إنشاء طلب مكرر، ويتم فقط استكمال التوزيع إذا كان الطلب غير موزع.
- إذا اختار قسمًا مختلفًا، يتم إغلاق طلب الخدمة المفتوح السابق بسبب تغيير القسم، مع الحفاظ على العميل والمحادثات والرسائل والملاحظات والسجل.
- يتم إنشاء طلب خدمة جديد بالحالة «عميل جديد» وإعادة توزيعه حسب قواعد القسم الجديد.
- عند التحويل إلى التمويل، يتم توزيع مندوب التمويل والكول سنتر طبقًا للقواعد الحالية.
- لا يتم حذف أي بيانات تاريخية ولا تغيير أي منطق خارج مسار إعادة التصنيف.
