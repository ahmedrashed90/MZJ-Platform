# Release checklist v1.9.3

## Scope
- إصلاح إرسال قوالب واتساب/مرسال فقط.
- لا تعديل على منطق العملاء أو التوزيع أو الحالات أو وكيل صندوق الوارد.
- لا تعديل على Worker واتساب/مرسال.

## Root cause
- قاعدة البيانات قد تحتوي على `template_send_url` قديم لمسار غير موجود.
- واجهة v1.9.2 تعرض مسار النص الموحد، لكن كود الإرسال كان يفضّل `template_send_url` عند القوالب.

## Fix
- واتساب/مرسال يستخدمان مسارًا موحدًا للنص والقالب.
- تهيئة قاعدة البيانات توحّد الحقول القديمة على المسار الفعلي.
- إضافة فحص بناء مخصص يمنع رجوع تفضيل مسار القالب القديم.

## Validation
- `npm run build`
- فحص TypeScript
- فحوص CRM الحالية
- `check-mersal-unified-send-v193.mjs`
- بناء نسخة ZIP نظيفة ثم إعادة `npm ci` و`npm run build`
